# 🤖 AI-Powered Chatbot

## 📌 Project Overview

The AI-Powered Chatbot is a web-based application designed to simulate human-like conversation and provide automated responses to user queries. It can handle basic greetings as well as answer general questions using external knowledge sources.

---

## 🚀 Features

* 💬 Interactive chatbot interface
* 🤖 Automated responses for user queries
* 🌐 Fetches real-time information using Wikipedia
* 🧠 Basic Natural Language Processing (NLP) support
* 🗂️ Stores chat history using SQLite database
* ⚡ Fast and lightweight web application

---

## 🛠️ Technologies Used

* **Python** – Core programming language
* **Flask** – Backend web framework
* **SQLite** – Database for storing chat logs
* **HTML, CSS, JavaScript** – Frontend interface
* **Wikipedia API** – For general knowledge responses

## 💡 How It Works

* The user enters a message in the chatbot interface
* The backend processes the input
* For basic queries (like greetings), predefined responses are returned
* For general topics, the chatbot fetches data from Wikipedia
* The response is displayed instantly to the user
